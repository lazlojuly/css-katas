z-index

  • specifies the stack order of an element
  • z-index only works on positioned elements (position:absolute, position:relative, or position:fixed).
  • default: auto (Sets the stack order equal to its parents.)
  • options: auto|number|initial|inherit;
  • all elements have z-index: 0 by default
[normal static element]

width: 300px

height: 300px

[z-index element]

width: 600px

height: 200px

position: fixed;

top:300px; left:150px;

z-index: -1;
